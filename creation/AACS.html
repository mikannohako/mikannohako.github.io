AACS - 自動出席確認システム

Automatic Attendance Confirmation System

info

プロジェクト概要

AACSは、Python で開発した自動出席確認システムです。中学校の部活動において、 従来の手書きによる出席管理の非効率性を解決するために開発しました。 QRコード認証と顔認証の2つの方式に対応し、柔軟な運用が可能です。

emoji_objects

開発の動機

  • 手書きの出席簿管理の手間を削減したい
  • 出席データの集計・分析を自動化したい
  • 不正な代理出席を防止したい
  • リアルタイムで出席状況を把握したい
settings

主な機能

qr_code_scanner

QRコード認証

個別に発行されたQRコードをスキャンして出席を記録

face

顔認証システム

カメラで顔を読み取り、自動的に本人確認と出席記録

analytics

データ集計

出席データを自動集計し、CSV形式で出力可能

speed

高速処理

瞬時に認証を完了し、スムーズな出席確認を実現

code
